# ADR-0008 — 화면 네비게이션 방식

| 항목 | 내용 |
|------|------|
| **상태** | ✅ Accepted |
| **결정일** | 2026.06.01 |

## 결정
**React Navigation** 대신 **자체 상태 기반 네비게이션**을 사용한다.

## 대안
- React Navigation
- Expo Router

## 이유
- 현재 화면 수가 적어 과도한 라이브러리 불필요
- 의존성 충돌 문제 회피
- useState로 충분히 관리 가능

## 결과
라이브러리 없이 홈·결과·히스토리 화면 전환 구현
