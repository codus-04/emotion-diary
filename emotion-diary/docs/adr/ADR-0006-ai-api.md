# ADR-0006 — AI 감정 분석 API 선택

| 항목 | 내용 |
|------|------|
| **상태** | ✅ Accepted |
| **결정일** | 2026.06.01 |

## 결정
**Claude API** (Anthropic) 를 감정 분석에 사용한다.

## 대안
- OpenAI GPT-4
- Google Gemini
- 자체 감정 분류 모델

## 이유
- 한국어 감정 표현 이해도 높음
- 공감적 응답 생성 능력 우수
- Cloud Functions에서 안전하게 호출

## 결과
자연스러운 한국어 공감 메시지 생성 가능
