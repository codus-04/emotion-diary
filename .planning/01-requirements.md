# 01. Requirements (MoSCoW)

## 프로젝트명
AI 감정일기 (Emotion Diary)

## 작성일
2026-05-04

---

## Must Have (반드시 구현)
> 없으면 앱이 의미 없는 핵심 기능

| ID | 요구사항 | 설명 |
|---|---|---|
| M-01 | 텍스트 일기 입력 | 사용자가 텍스트로 감정 일기 작성 가능 |
| M-02 | 음성 입력 | 음성을 텍스트로 변환하여 일기 작성 가능 |
| M-03 | AI 감정 분석 | Claude API로 일기 내용의 감정 분석 및 태깅 |
| M-04 | 감정 저장 | 작성한 일기와 감정 데이터 로컬 저장 (Isar) |
| M-05 | 감정 그래프 | 주간/월간 감정 흐름 시각화 (fl_chart) |
| M-06 | 일기 목록 조회 | 날짜별 일기 목록 및 상세 내용 확인 |

---

## Should Have (가능하면 구현)
> 있으면 완성도가 높아지는 기능

| ID | 요구사항 | 설명 |
|---|---|---|
| S-01 | 주간 감정 리포트 | AI가 한 주 감정 패턴 요약 리포트 생성 |
| S-02 | 푸시 알림 | 매일 정해진 시간에 기록 리마인더 알림 |
| S-03 | 감정 태그 필터 | 특정 감정(기쁨/슬픔 등)으로 일기 필터링 |
| S-04 | 감정 인사이트 | "비 오는 날 우울감 증가" 같은 패턴 발견 |
| S-05 | 다크모드 지원 | 시스템 다크모드 연동 |

---

## Could Have (여유 있으면 구현)
> 있으면 좋지만 없어도 무방한 기능

| ID | 요구사항 | 설명 |
|---|---|---|
| C-01 | 명상 콘텐츠 추천 | 감정 상태에 따라 명상/음악 콘텐츠 추천 |
| C-02 | 감정 캘린더 | 월별 감정을 색상으로 표시하는 캘린더 뷰 |
| C-03 | 백업 / 복원 | Supabase 클라우드 백업 및 기기 간 동기화 |
| C-04 | 위젯 지원 | 홈 화면 위젯으로 오늘의 감정 빠른 기록 |

---

## Won't Have (이번 버전에서 제외)
> 범위를 벗어나거나 우선순위가 낮은 기능

| ID | 요구사항 | 제외 이유 |
|---|---|---|
| W-01 | SNS 공유 기능 | 감정 일기는 프라이버시가 중요, 공유 기능 부적합 |
| W-02 | 다국어 지원 | 초기 버전은 한국어만 지원 |
| W-03 | 웨어러블 연동 | Apple Watch 연동은 2.0 버전에서 고려 |
| W-04 | 커뮤니티 기능 | 소셜 기능은 핵심 가치와 맞지 않음 |

## 비기능 요구사항

| 항목 | 요구사항 |
|---|---|
