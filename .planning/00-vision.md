# 00. Product Vision

## 프로젝트 이름
AI 감정일기 (Emotion Diary)

## 한 줄 설명
매일 감정을 기록하면 AI가 패턴을 분석해 자기이해를 돕는 모바일 앱

---

## 배경 및 문제 정의
현대인들은 바쁜 일상 속에서 자신의 감정을 인식하고 돌아볼 시간이 부족하다.
기존 일기 앱은 단순 기록에 그치며, 감정 데이터를 분석해 인사이트를 제공하지 않는다.
사용자는 자신이 언제, 왜 특정 감정을 느끼는지 패턴을 파악하기 어렵다.

---

## 타겟 유저
- 주 타겟: 20~30대 자기계발에 관심 있는 직장인 / 대학생
- 부 타겟: 감정 조절이 어렵거나 번아웃을 경험하는 사람

---

## 핵심 가치 (Value Proposition)
> "그냥 일기가 아닌, 나를 이해하는 AI 파트너"

1. **쉬운 기록** — 텍스트 또는 음성으로 30초 안에 감정 기록
2. **AI 분석** — Claude API가 감정의 뉘앙스를 분석해 감정 레이블 자동 태깅
3. **패턴 시각화** — 주간/월간 감정 흐름을 그래프로 한눈에 확인
4. **인사이트 리포트** — "비 오는 날 우울감이 높아지는 패턴" 같은 맞춤 인사이트 제공

---

## 차별점
| 기존 일기 앱 | AI 감정일기 |
|---|---|
| 단순 텍스트 저장 | AI 감정 분석 + 자동 태깅 |
| 기록만 가능 | 패턴 분석 + 인사이트 제공 |
| 데이터 활용 없음 | 시계열 감정 그래프 시각화 |
| 음성 입력 없음 | 음성 → 텍스트 변환 지원 |

---

## 핵심 기능 요약
- 텍스트 / 음성 일기 입력
- Claude API 기반 감정 분석 및 태깅
- 감정 트렌드 그래프 (fl_chart)
- 주간 감정 리포트 자동 생성
- 푸시 알림 (매일 기록 리마인더)

---

## 성공 지표
- 7일 연속 기록 달성 사용자 비율
- 감정 분석 정확도 사용자 만족도
- 주간 리포트 열람률

---

## 기술 스택 요약
- Frontend: Flutter (iOS / Android)
- AI: Claude API (claude-haiku)
- DB: Isar (로컬), Supabase (백엔드)
- 상태관리: Riverpod
- CI/CD: GitHub Actions + Firebase App Distribution
